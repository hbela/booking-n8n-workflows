# Booking n8n Workflows

This repository contains n8n workflows for the booking system.

## Structure

- `workflows/`: Contains the workflow JSON files.
- `scripts/`: Helper scripts for deployment and maintenance.
- `tests/`: Unit and integration tests for workflows.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Testing

Run tests with:
```bash
npm test
```

## Deployment

Deploy workflows using the helper script:
```bash
npm run deploy
```
